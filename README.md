## 🤖 Introduction

Booking App is a modern mobile application built with React Native.  
It allows users to explore properties, view detailed information, and make bookings directly from their mobile devices.

---

## ⚙️ Tech Stack

- Expo  
- React Native  
- TypeScript  
- Appwrite  
- NativeWind (Tailwind CSS)  

---

## 🔋 Features

👉 Browse available properties  
👉 View detailed property information  
👉 Booking system  
👉 User authentication (Google / Email)  
👉 Profile management  
👉 Dynamic data fetching
