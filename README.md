## 🤖 Introduction

This is a full-stack mobile application built with **React Native**, designed to provide a seamless experience for exploring and booking properties.

The project showcases practical development skills such as:
- UI/UX design  
- API integration  
- State management  
- Real-time data handling  

---

## ⚙️ Tech Stack

- ⚛️ React Native  
- 🚀 Expo  
- 🟦 TypeScript  
- 🔥 Appwrite (Backend as a Service)  
- 🎨 NativeWind (Tailwind CSS)  

---

## 🔋 Features

- 🔍 Browse and search properties  
- 🏠 View detailed property information  
- 📅 Booking functionality  
- 🔐 Authentication (Google / Email)  
- 👤 User profile management  
- ⚡ Fast and responsive UI  
- 🔄 Dynamic data fetching

## 🤸 Quick Start

### 📌 Prerequisites

Make sure you have:

- Git  
- Node.js  
- npm  

---

### 📥 Clone Project

```bash
git clone https://github.com/dinhminhtaicv-cyber/Booking.App.git
cd Booking.App
npm install
git clone https://github.com/dinhminhtaicv-cyber/Booking.App.git
cd Booking.App
📦 Install dependencies
npm install
⚙️ Setup Environment Variables

Tạo file .env trong root:

EXPO_PUBLIC_APPWRITE_ENDPOINT=
EXPO_PUBLIC_APPWRITE_PROJECT_ID=
EXPO_PUBLIC_APPWRITE_DATABASE_ID=
EXPO_PUBLIC_APPWRITE_PROPERTIES_COLLECTION_ID=
EXPO_PUBLIC_APPWRITE_AGENTS_COLLECTION_ID=
EXPO_PUBLIC_APPWRITE_REVIEWS_COLLECTION_ID=
▶️ Run app
npx expo start
📂 Project Structure
Booking.App/
│── app/            # Screens
│── components/     # Reusable components
│── lib/            # API & logic
│── assets/         # Images, icons
│── constants/      # Static data
│── App.tsx
